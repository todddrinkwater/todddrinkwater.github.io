Tech Blog: Brackets, braces, and more...

January 27, 2017

Explain the meaning of...

Parentheses ( )

Parentheses are used for three main things. These are:

  • To contain parameters within functions
  • To contain conditional statements
  • To group items together when applying the order of operations.

In Functions, brackets hold parameters inside of them. These parameters can then be replaced with arguments when the function is called and the function will implement this into the course of actions it has been programmed to take. By having these parameters, it makes the function more reusable as it has the option of passing different arguments through it everytime it is called.

With conditional statements (such as the if statement), the statements that are placed between the parentheses are evaluate to see whether they return true or not. If they do return a true value, then the code following inside the braces will run.

Following the Order of Operations, statements or calculations that are placed within the brackets will be calculated before anything else. For example, in the statement a * (b + c), b and c will be calculated first before multiplying by a. The logic of the Order of Operation is (working from left to right):

  1. Parentheses
  2. Exponents
  3. Multiplication and Division
  4. Addition and Subtraction

So by placing b and c inside brackets, we were able to move b + c to the front of the queue.

Brackets [ ]

Braces are most often used to contain data from arrays and to also access that data using zero-based indexing. For instance, an array defined as myArr may contain the data [a, b, c] (e.g. var myArr = [a, b, c]). If I wanted to access a from the array I would use the statement myArr[0]

Braces { }

Braces are used within code to either define object literals or enclose blocks of code for such things as functions or loops. Within object literals, you find properties and values that have been defined. However, within functions or loops, you will find a set of code, giving instructions for what to do when that particular function is called or loop is run.

Quotation Marks " " and ' '

Quotation marks are used within Javascript to define something as a string. Numbers can also be defined as strings if placed between quotation marks. Whether you choose to use single or double quotation marks is of personal preference to the programmer. One important thing to remember when using quotation marks within a string, to place a \ before each quotation mark ( e.g. alert("You\'re welcome!") ). This allows for the console to distinguish between the quotation marks used to define a string and the quotation marks for the user to see on the webpage.
